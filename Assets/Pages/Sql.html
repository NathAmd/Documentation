Documentation Complète sur SQL

Cette documentation couvre les commandes et fonctionnalités globales de SQL, ainsi que les spécificités pour MySQL, SQL Server, SQLite, et plus encore. Une section spéciale est dédiée aux structures de données.

Commandes SQL de Base

Les commandes SQL de base permettent de lire, insérer, modifier et supprimer des données dans une base de données.

  • SELECT: 
    
-- Sélectionner toutes les colonnes d'une table
SELECT * FROM table_name;

-- Sélectionner des colonnes spécifiques
SELECT column1, column2 FROM table_name;
    Explication

    La commande SELECT est utilisée pour récupérer des données d'une base de données.

  • INSERT INTO: 
    
-- Insérer des données dans une table
INSERT INTO table_name (column1, column2) VALUES (value1, value2);
    Explication

    La commande INSERT INTO est utilisée pour ajouter de nouvelles lignes dans une table.

  • UPDATE: 
    
-- Mettre à jour des données dans une table
UPDATE table_name SET column1 = value1, column2 = value2 WHERE condition;
    Explication

    La commande UPDATE est utilisée pour modifier des données existantes dans une table.

  • DELETE: 
    
-- Supprimer des données d'une table
DELETE FROM table_name WHERE condition;
    Explication

    La commande DELETE est utilisée pour supprimer des lignes d'une table.

  • CREATE TABLE: 
    
-- Créer une nouvelle table
CREATE TABLE table_name (
    column1 datatype,
    column2 datatype,
    ...
);
    Explication

    La commande CREATE TABLE est utilisée pour créer une nouvelle table dans la base de données.

  • DROP TABLE: 
    
-- Supprimer une table
DROP TABLE table_name;
    Explication

    La commande DROP TABLE est utilisée pour supprimer une table de la base de données.

Fonctions SQL

Les fonctions SQL permettent de manipuler et d'analyser les données de manière plus avancée.

  • Fonctions de chaîne: 
    
-- Convertir une chaîne en majuscules
SELECT UPPER(column_name) FROM table_name;

-- Extraire une sous-chaîne
SELECT SUBSTRING(column_name, start, length) FROM table_name;
    Explication

    Les fonctions de chaîne permettent de manipuler les chaînes de caractères.

  • Fonctions numériques: 
    
-- Calculer la somme d'une colonne
SELECT SUM(column_name) FROM table_name;

-- Calculer la moyenne d'une colonne
SELECT AVG(column_name) FROM table_name;
    Explication

    Les fonctions numériques permettent de réaliser des calculs sur les données numériques.

  • Fonctions de date: 
    
-- Obtenir la date actuelle
SELECT CURRENT_DATE;

-- Ajouter des jours à une date
SELECT DATE_ADD(column_name, INTERVAL 10 DAY) FROM table_name;
    Explication

    Les fonctions de date permettent de manipuler les valeurs de date et d'heure.

  • Fonctions d'agrégation: 
    
-- Compter le nombre de lignes
SELECT COUNT(*) FROM table_name;

-- Trouver la valeur maximale
SELECT MAX(column_name) FROM table_name;
    Explication

    Les fonctions d'agrégation permettent de réaliser des calculs sur un ensemble de valeurs.

Structures de Données en SQL

Les structures de données en SQL sont essentielles pour organiser et optimiser les bases de données.

  • Types de données: 
    
-- Types de données courants
INTEGER, VARCHAR(n), DATE, BOOLEAN, FLOAT
    Explication

    Les types de données déterminent le type de données pouvant être stocké dans une colonne.

  • Clés primaires et étrangères: 
    
-- Définir une clé primaire
CREATE TABLE table_name (
    column1 datatype PRIMARY KEY,
    column2 datatype
);

-- Définir une clé étrangère
CREATE TABLE table_name (
    column1 datatype,
    column2 datatype,
    FOREIGN KEY (column1) REFERENCES other_table(column)
);
    Explication

    Les clés primaires et étrangères sont utilisées pour garantir l'intégrité des données et établir des relations entre les tables.

  • Index: 
    
-- Créer un index
CREATE INDEX index_name ON table_name (column_name);

-- Créer un index unique
CREATE UNIQUE INDEX index_name ON table_name (column_name);
    Explication

    Les index améliorent les performances des requêtes en permettant un accès plus rapide aux données.
